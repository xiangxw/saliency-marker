## Install
#### Ubuntu(14.04, 14.10, 15.04):

```
    sudo apt-add-repository -y ppa:xiangxw5689/saliency-marker
    sudo apt-get update
    sudo apt-get -y install saliency-marker
```

## Compile
Install CMake and Qt5, then:

```
    mkdir build
    cd build
    cmake ..
    make
```
